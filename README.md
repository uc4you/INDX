# Bondtech INDX

The Bondtech INDX is a modular, mechanical tool-changing platform for FDM/FFF 3D printers. Unlike traditional single-tool setups or filament-swapping
systems, Bondtech INDX physically swaps out entire tool assemblies—including nozzle, heat break, and filament path—enabling true multi-material,
multi-nozzle, and multi-application 3D printing. It integrates a proprietary mechanical extruder, the Dynamic Dual Drive, that adapts to filament resistance
without sensors or electronics.
